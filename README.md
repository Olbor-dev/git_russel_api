# Capitainerie de port Russel - Gestion de catway (Exercice CEF)

Création d'une application web node.js pour la gestion des réservations de catways.

## Installation

1.  ** Cloner le dépot **
    ```bash
    git clone https://github.com/Olbor-dev/git_russel_api.git

2.  ** Se déplacer dans le dossier créé **
    ```bash
    cd git_russel_api

3.  ** Installer NPM **
    ```bash
    npm install

4.  ** Lancer le projet (en dev pour le moment) **
    ```bash
    npm run dev

5.  ** Ouvre le serveur local **
    http://localhost:3000
